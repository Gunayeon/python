## 버림 나눗셈
```
#floor division (버림 나눗셈)
print(7//2)

print(8//3)
print(8.0//3)
print(8.0//3.0)
```
> 버림 나눗셈은 연산자중 하나라도 소수형이면 소수형으로 결과가 출력됨

```
#round 반올림
print(round(3.1415926535))
print(round(3.1415926535,2))
```
> round는 반올림 연산자인데 3번째줄 같은경우 ,2는 소수점 두번째자리까지 나타낸다는 의미이다

## 문자열
문자열:'파이썬', "파이썬"->파이썬이라는 문자열완성
문자열 연산:문자열 연결
```
"hello"+"world":"helloworld"
"hello"*3="hello"+"hello"+"hello"=hellohellohello
```
```
print("hello"+"hello"+"hello")
print("hello"*3)
print("3"+"5")
```
+) 따옴표를 출력하고 싶을 때
```
## 따옴표 의도

```

## 형 변환
숫자형->숫자형
```
print(int(3.8)) #정수형 변환
print(float(3)) #소수형 변환
```
문자열->숫자형
```
print(int("2")+int("5"))
print(float("1.1")+float("2.5"))
```
숫자형->문자열
```
print(str(2)+str(5))
# example
age=7
print("제 나이는 "+str(age)+"살입니다.")
```
* 주의:문자열에서 숫자형은 형변환 불가

## format을 이용한 문자열 포맷팅
```
# 오늘은 2019년 10월 29일 입니다.
```
형변환
```
print("오늘은 "+str(year)+"년 "+str(month)+"월 "+str(day)+"일입니다.")#형변환 이용
```
> 일일이 형변환하기 번거로움

format메소드 이용
```
#1번째 방법
print("오늘은 {}년 {}월 {}일입니다.".format(year,month,day)) #format메소드 이용
```
```
#2번째 방법
date_string="오늘은 {}년 {}월 {}일입니다."
print("date_string".format(year,month,day))
```
## format다루기
파라미터를 이용함
ex)\
```
#저는 유재석, 박지성, 빌 게이츠를 좋아합니다.
print("저는 {1}, {0}, {2}를 좋아합니다!".format("박지성", "유재석", "빌 게이츠"))
# 0:1번째 ->"박지성"을 가리킴
# 1:2번째 ->"유재석"을 가리킴
# 2:3번째 ->"빌 게이츠"를 기리킴
```
숫자형 format
```
num_1=1
num_2=3
print("{0} 나누기 {1}은 {2:.2f}입니다.".format(num_1, num_2, num_1/num_2))
```
2번 중괄호의 숫자를 소수점 n자리까지 표현하려면 :.nf라고 나타낸다.
## 불린(Boolean)
```
#불린(Boolean)
#and
print(True and True)
print(True and False)
print(False and True)
print(False and False)

#or
print(False or False)
print(False or False)
print(False or False)
print(False or False)

#not
print(not True)
print(not False)

#연산식
print(2>1)
print(2<1)
print(3>=2)
print(3<=2)
print(2==2)
print(2!=2)

#문자열
print("Hello"=="Hello")
print("Hello"!="Hello")

#문자열과 연산식
print(2>1 and "Hello"=="Hello")

#2중부정
print(not not True)
print(not not False)

#괄호
print(7==7 or (4<3 and 12>10))

#변수 있을떄
x=3
print(x>4 or not (x<2 or x==3))
```
## Type함수
