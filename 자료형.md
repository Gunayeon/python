# 자료형
## 정수형 연산
```
# 덧셈
print(4+7)

# 뺄셈
print(2-4)

# 곱셈
print(5*3)

# 나머지
print(7%3)

# 거듭제곱
print(2**3)
```

## 소수형 연산
```
# 덧셈
print(4.0+7.0)

# 뺄셈
print(2.0-4.0)

# 곱셈
print(5.0*3.0)

# 나머지
print(7.0%3.0)

# 거듭제곱
print(2.0**3.0)

```
