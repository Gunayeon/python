## 모듈
모듈을 사용하는 이유:한 파일에 함수와 실행부분을 넣게 되면 코드가 너무 길어질 수 있음
> 따라서 실행부분은 따로 파일을 만들어서 함수파일을 따로 호출함

호출방법
* 새 파일 생성 후 호출
ex) 함수 파일이름:calculator
    add, sub, mul, div 함수가 존재
```
import calculator #calculator파일 호출

print(calculator.add(2,5))
print(calculator.mul(3,5))
```
* 호출 시 파일이름을 임의로 지정할 수 있음
```
import calculator as calc #호출시 calc로 작성하여 호출

print(calc.add(2,5))
print(calc.mul(3,5))
```
* 사용할 함수만 호출하고 싶을 때
```
from calculator import add, mul

print(add(2,5))
print(mul(3,4))
```

### random모듈
randint 함수 \
randint는 두 수 사이의 어떤 랜덤한 정수를 리턴하는 함수.
randint(a, b)를 하면, a ≤ N ≤ b를 만족하는 어떤 랜덤한 정수 N을 리턴.
```
import random

print(random.randint(1, 20))
print(random.randint(1, 20))
print(random.randint(1, 20))
print(random.randint(1, 20))
print(random.randint(1, 20))
```
uniform 함수 \
uniform은 두 수 사이의 랜덤한 소수를 리턴하는 함수. 
uniform(a, b)를 하면, a ≤ N ≤ b를 만족하는 어떤 랜덤한 소수 N을 리턴.
```
import random

print(random.uniform(0, 1))
print(random.uniform(0, 1))
print(random.uniform(0, 1))
print(random.uniform(0, 1))
print(random.uniform(0, 1))
```

### datetime모둘
```
import datetime
#datetime값 생성
pi_day=datetime.datetime(2022,7,6)
print(pi_day)
print(type(pi_day))

#오늘 날짜
pi_day=datetime.datetime(2022,7,6,11,51,45)
print(pi_day)
print(type(pi_day))

#timedelta
today=datetime.datetime.now()
pi_day=datetime.datetime(2022,7,6,11,51,45)
print(today-pi_day)
print(type(today-pi_day))

today=datetime.datetime.now()
my_timedelta=datetime.timedelta(days=5,hours=3,minutes=10,seconds=50)
print(today)
print(today+my_timedelta)

#datetime해부하기
today=datetime.datetime.now()

print(today)
print(today.year)
print(today.month)
print(today.day)
print(today.hour)
print(today.minute)
print(today.second)
print(today.microsecond)

#datetime 포맷팅
today=datetime.datetime.now()

print(today)
print(today.strftime("%A, %B %dth %Y"))
```

## 파일 읽기
텍스트 파일을 파이썬에 불러와 읽는거
* 텍스트파일을 실행파일과 같은 폴더에 옮겨온다.
```
with open('data/chicken.txt', 'r'') as f:
             파일명           읽기      변수 f에 저장
```
> 출력이 되긴하나 리스트도 아니고 암튼 뭔지모름\
> 따라서 텍스트파일에 있는 내용을 한줄한줄 출력하고 싶을때는 반복문사용

```
with open('data/chicken.txt', 'r', encoding='UTF-8') as f:
    for line in f:
        print(line)
```
