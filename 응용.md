## 모듈
모듈을 사용하는 이유:한 파일에 함수와 실행부분을 넣게 되면 코드가 너무 길어질 수 있음
> 따라서 실행부분은 따로 파일을 만들어서 함수파일을 따로 호출함

호출방법
* 새 파일 생성 후 호출
ex) 함수 파일이름:calculator
    add, sub, mul, div 함수가 존재
```
import calculator #calculator파일 호출

print(calculator.add(2,5))
print(calculator.mul(3,5))
```
* 호출 시 파일이름을 임의로 지정할 수 있음
```
import calculator as calc #호출시 calc로 작성하여 호출

print(calc.add(2,5))
print(calc.mul(3,5))
```
* 사용할 함수만 호출하고 싶을 때
```
from calculator import add, mul

print(add(2,5))
print(mul(3,4))
```

### random모듈
randint 함수 \
randint는 두 수 사이의 어떤 랜덤한 정수를 리턴하는 함수.
randint(a, b)를 하면, a ≤ N ≤ b를 만족하는 어떤 랜덤한 정수 N을 리턴.
```
import random

print(random.randint(1, 20))
print(random.randint(1, 20))
print(random.randint(1, 20))
print(random.randint(1, 20))
print(random.randint(1, 20))
```
uniform 함수 \
uniform은 두 수 사이의 랜덤한 소수를 리턴하는 함수. 
uniform(a, b)를 하면, a ≤ N ≤ b를 만족하는 어떤 랜덤한 소수 N을 리턴.
```
import random

print(random.uniform(0, 1))
print(random.uniform(0, 1))
print(random.uniform(0, 1))
print(random.uniform(0, 1))
print(random.uniform(0, 1))
```

### datetime모둘
```
import datetime
#datetime값 생성
pi_day=datetime.datetime(2022,7,6)
print(pi_day)
print(type(pi_day))

#오늘 날짜
pi_day=datetime.datetime(2022,7,6,11,51,45)
print(pi_day)
print(type(pi_day))

#timedelta
today=datetime.datetime.now()
pi_day=datetime.datetime(2022,7,6,11,51,45)
print(today-pi_day)
print(type(today-pi_day))

today=datetime.datetime.now()
my_timedelta=datetime.timedelta(days=5,hours=3,minutes=10,seconds=50)
print(today)
print(today+my_timedelta)

#datetime해부하기
today=datetime.datetime.now()

print(today)
print(today.year)
print(today.month)
print(today.day)
print(today.hour)
print(today.minute)
print(today.second)
print(today.microsecond)

#datetime 포맷팅
today=datetime.datetime.now()

print(today)
print(today.strftime("%A, %B %dth %Y"))
```

## 파일 읽기

```
with open('data/chicken.txt', 'r'') as f:
             파일명           읽기      변수 f에 저장
```
> 출력이 되긴하나 리스트도 아니고 암튼 뭔지모름\

## split
.split():파라미터 기준으로 문자열을 나눔
```
my_string="1. 2. 3. 4. 5. 6"
print(my_string.split(". ")) #파라미터 기준으로 문자열을 나눔
```
-> ['1', '2', '3', '4', '5', '6']로 출력.

```
full_name="Kim, Yuna"
print(full_name.split(", "))
```
-> ['Kim', 'Yuna']로 출력.

변수로 저장해 인덱스로 나누어 출력도 가능
```
name_data=full_name.split(", ") 
first_name=name_data[0]
last_name=name_data[1]
```
문자열 내 화이트스페이스기준으로 나눌수도 있음
```
print("   \n\n  2 \t  3  \n  5 7 11  \n\n".split()) 
```
-> ['2', '3', '5', '7', '11']로 출력

인덱스로 나누어서 연산도 가능함
```
numbers="   \n\n  2 \t  3  \n  5 7 11  \n\n".split()
print(int(numbers[0])+int(numbers[1]))  
```
