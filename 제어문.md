# 제어문
## while문
while:무언가를 반복하기 위해 사용함
```
while:조건문(불린값으로 계산되는 식)
   시행부분(우리가 반복하고 싶은 명령)
```
조건문이 false가 될때까지 시행부분이 반복됨

ex)"나는 잘생겼다!"를 3번 반복
```
i=1
while i<=3:
    print("나는 잘생겼다!")
    i+=1
```
## if문
```
if 조건부분:(불린값으로 계산되는 식)
    수행부분
    
```
ex)
```
temperature=16
if temperature<=10:
    print("자켓을 입는다.")
else:
    print("자켓을 입지 않는다.")
```
## elif문
else if와 의미가 같음

## break/continue문
* break:while문의 조건부분과 상관없이 반복문에서 나오고 싶으면 사용함.
* continue:현재 진행되고 있는 수행부분을 중단하고 바로 조건 부분을 확인하고 싶으면 사용

## for문
for도 반복문이다.
while과 달리 조건이 없다.
```
my_list=[2,3,5,7,11]

for number in my_list: #my_list의 요소가 number로 들어간다
    print(number)
```
