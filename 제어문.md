# 제어문
## while문
while:무언가를 반복하기 위해 사용함
```
while:조건문(불린값으로 계산되는 식)
   시행부분(우리가 반복하고 싶은 명령)
```
조건문이 false가 될때까지 시행부분이 반복됨

ex)"나는 잘생겼다!"를 3번 반복
```
i=1
while i<=3:
    print("나는 잘생겼다!")
    i+=1
```
## if문
```
if 조건부분:(불린값으로 계산되는 식)
    수행부분
    
```
ex)
```
temperature=16
if temperature<=10:
    print("자켓을 입는다.")
else:
    print("자켓을 입지 않는다.")
```
