# 리스트
## 리스트
* 정수형 리스트 선언
```
numbers=[2,3,5,7,11,13] #이 안에 있는 값은 리스트안의 요소
```

* 문자열 리스트 선언
```
names=["윤수","혜린","태호","영훈"]
```

* 인덱싱
리스트 인덱스의 범위는 만약 리스트 안의 요소가 n개라고 가정했을때, -n~n-1`
```
print(names[0])
print(numbers[1]+numbers[3])
print(numbers[-1])#맨오른쪽으로 부터 첫번째 인덱스
print(numbers[-2])#맨오른쪽으로 부터 두번째 인덱스
print(numbers[-6])#맨 앞 인덱스
```
* 리스트 슬라이싱
리스트 요소의 일부만 다루는거

```
print(numbers[0:4]) #0~3번 인덱스 출력
print(numbers[2:]) #2번~마지막 인덱스출력
print(numbers[:3]) #첫번째~3번인덱스 출력

#새로운 리스트를 지정하는 방법도 있음
new_list=numbers[:3] #[2,3,5]
print(new_list[2])
```

* 리스트 요소 바꾸기
```
numbers[0]=7
print(numbers)

numbers[0]=numbers[0]+numbers[1]
print(numbers)
```
