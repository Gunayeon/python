# 리스트
## 리스트
* 정수형 리스트 선언
```
numbers=[2,3,5,7,11,13] #이 안에 있는 값은 리스트안의 요소
```

* 문자열 리스트 선언
```
names=["윤수","혜린","태호","영훈"]
```

* 인덱싱
리스트 인덱스의 범위는 만약 리스트 안의 요소가 n개라고 가정했을때, -n~n-1`
```
print(names[0])
print(numbers[1]+numbers[3])
print(numbers[-1])#맨오른쪽으로 부터 첫번째 인덱스
print(numbers[-2])#맨오른쪽으로 부터 두번째 인덱스
print(numbers[-6])#맨 앞 인덱스
```
* 리스트 슬라이싱
리스트 요소의 일부만 다루는거

```
print(numbers[0:4]) #0~3번 인덱스 출력
print(numbers[2:]) #2번~마지막 인덱스출력
print(numbers[:3]) #첫번째~3번인덱스 출력

#새로운 리스트를 지정하는 방법도 있음
new_list=numbers[:3] #[2,3,5]
print(new_list[2])
```

* 리스트 요소 바꾸기
```
numbers[0]=7
print(numbers)

numbers[0]=numbers[0]+numbers[1]
print(numbers)
```
## 리스트 함수
* 리스트 요소의 개수를 구하는 함수:len
```
numbers=[]
len(numbers) #리스트 요소의 개수를 구하는 함수,length의 줄임말
print(len(numbers))
```

* 리스트에 요소 추가:append
```
numbers.append(5)
numbers.append(8) #append를 하면 리스트의 가장 오른쪽에 요소 추가
print(numbers)
print(len(numbers))
```

* 리스트 요소 삭제:del
```
numbers=[2,3,5,7,11,13,17,19]
del numbers[3] #3번 인덱스 삭제
print(numbers)
```

* 리스트의 원하는 위치에 요소 삽입:insert
```
numbers=[2,3,5,7,11,13,17,19]
numbers.insert(4, 37) #4번 인덱스에 37삽입
print(numbers)
```
## 리스트 정렬
* sorted 정렬:기존 리스트는 건드리지 않고, 정렬된 새로운 리스트를 리턴
* sort:아무것도 리턴하지 않고, 기존 리스트를 정렬
```
#주어진 리스트
numbers=[19,13,12,2,5,3,11,7,17]

#sorted
new_list=sorted(numbers)
print(new_list) #정방향

new_list=sorted(numbers,reverse=True)
print(new_list) #역방향

#sort
numbers.sort()
print(numbers) #정방향

numbers.sort(reverse=True)
print(numbers) #역방향
```

## 리스트에서 값의 존재 확인하기
* 리스트에서 value를 찾으면 true 리턴
* 리스트에서 value를 찾지 못하면 False 리턴

```
def in_list(some_list,value):
    i=0
    while i<len(some_list):
        if some_list[i]==value:
            return  True
        i=i+1

    return  False

primes=[2,3,5,7,11,13,17,19,23]
print(in_list(primes,7))
print(in_list(primes,12))
```
값이 없는지 확인하려면 in 앞에 not을 붙인다.
```
primes = [2, 3, 5, 7, 11, 13, 17, 19, 23]
print(7 not in primes)
print(12 not in primes)
```

# Nested List
* 리스트 안의 또 다른 리스트

```
#세 번의 시험을 보는 수업
grades = [[62, 75, 77], [78, 81, 86], [85, 91, 89]]

print(grades[0]) #첫 번째 학생의 성적

print(grades[2]) #세 번째 학생의 성적

print(grades[0][0]) #첫 번째 학생의 첫 번째 시험 성적

print(grades[2][1]) #세 번째 학생의 두 번째 시험 성적

print((grades[0][0] + grades[1][0] + grades[2][0]) / 3) #첫 번째 시험의 평균
```

## index메소드
