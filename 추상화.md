## return/print 차이
```
def print_square(x):
    print(x*x)

def get_square(x):
    return x*x

#return/print 차이

print_square(3)
print(get_square(3)) #x에 3을 대입해 3의 제곱을 반환받아 그것을 출력
print(print_square(3)) #x에 3을대입해 3의 제곱을 출력, 그러나 함수가종료되지 않아 한번더 실행되는데 return이 없어 
                        #반환을 하지 않기 때문에 None 출력
```
## 옵셔널 파라미터
```
def myself(name,age,nationality="한국"):
    print("내 이름은 {}".format(name))
    print("나이는 {}살".format(age))
    print("국적은 {}".format(nationality))

myself("코드잇",1,"미국")
print()
myself("코드잇",1) #옵셔널 파라미터:파라미터에 기본값 설정가능, 설정하면 함수를 호출할 때 파라미터에 값을 안넘겨줘도 됨

```
> 옵셔널 파라미터는 꼭 마지막에 써야함

# scope

scope:변수가 사용가능한 범위

* 로컬변수: 함수내에서 정의된 변수, 함수내에서만 사용할 수 있음
```
def my_function():
    x=3 #로컬변수,my_function안에서만 지정됨
    print(x)

my_function()
print(x) #지정되지 않은변수
```
* 글로벌 변수:프로그램내에서 정의된 변수, 프로그램 전체 안에서 사용가능
```
x=2

def my_function():
    print(x)
    
my_function()
print(x)
```
* 로켤 변수/글로벌 변수 둘 다 있을때
```
x=2 #글로벌 변수

def my_function():
    x=3 #로컬 변수
    print(x) #함수내라서 로컬 변수 출력

my_function()
print(x) #글로벌 변수 출력
```
* 파라미터는 로컬 변수
