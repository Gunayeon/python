# 추상화
## '='의 의미
등호의 의미가 아닌 값을 지정하는 지정연산자이다.
## 함수 실행순서
```
def hello():
    print("Hello!")
    print("Welcome to Codeit!")

print("함수 호출 전")
hello() #함수 실행
print("함수 호출 후")

def square(x):
    return x*x

print("함수 호출 전")
print(square(3)+square(4)) #함수 실행
print("함수 호출 후")
```
## return/print 차이
```
def print_square(x):
    print(x*x)

def get_square(x):
    return x*x

#return/print 차이

print_square(3)
print(get_square(3)) #x에 3을 대입해 3의 제곱을 반환받아 그것을 출력
print(print_square(3)) #x에 3을대입해 3의 제곱을 출력, 그러나 함수가종료되지 않아 한번더 실행되는데 return이 없어 
                        #반환을 하지 않기 때문에 None 출력
```
